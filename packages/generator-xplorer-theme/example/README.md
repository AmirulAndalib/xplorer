# Example of Xplorer custom color theme

These are examples generated by the `generator-xplorer-theme`. For more detailed information, please visit [this](https://xplorer.vercel.app/docs/customization/theme/).

## Development

To start this on development, go into a directory and run following command:

```bash
yarn start
```

## Production

To build this theme, go into a directory and run following command and the color theme will be installed on Xplorer.

```bash
yarn build
```
